# Loops, Conditions and Parsing

This project contains Bash scripts demonstrating the use of loops, conditional statements, and file parsing.

## Files Description
- 1-for_best_school: Uses for loop
- 2-while_best_school: Uses while loop
- 3-until_best_school: Uses until loop
- 4-if_9_say_hi: Uses if statement with while loop
- 5-4_bad_luck_8_is_your_chance: Uses if-elif-else
- 6-superstitious_numbers: Uses case statement
- 7-clock: Nested while loops for clock display
- 8-for_ls: Uses for loop with cut command
- 9-to_file_or_not_to_file: Uses file test operators
- 10-fizzbuzz: FizzBuzz implementation
- 11-read_and_cut: Reads and parses /etc/passwd
- 12-tell_the_story_of_passwd: Creates story from /etc/passwd
- 13-lets_parse_apache_logs: Parses Apache logs with awk
- 14-dig_the-data: Analyzes Apache log data
